SpeakEasy is an extension for Google Chrome that allows the user to navigate the Internet and the browser with their voice.
Developed by Jacob Beckerman and Jacob Wright
Champlain College ITS Division
Computer Science & Innovation Capstone 2019

STEPS FOR INSTALLATION:
1. In Chrome, navigate to chrome://extensions
2. In the top right corner of the page, turn developer mode on.
3. In the top left corner of the page, click "Load unpacked"
4. Navigate to the root folder of the extension and click "Select folder"
5. See that SpeakEasy is enabled in the list of extensions, open a new tab, and navigate away from the instructional page

Now you're ready to use SpeakEasy!
